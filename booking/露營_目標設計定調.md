# 森日奶油露營｜目標設計定調

## 1. 設計定位

整體風格定義為：

> 療癒系文青奶油風 × 日系露營選物 × 北歐自然生活感

核心不是「戶外冒險」，而是「將露營融入日常生活」。

設計感受應包含：

- 溫柔而不甜膩
- 自然而不粗獷
- 文青而不過度裝飾
- 奶油色但不泛黃
- 綠色低飽和，不使用螢光感
- 商城功能完整，但降低促銷壓迫感

品牌關鍵字：

```text
慢生活、森林、晨光、亞麻、原木、手沖咖啡、
週末露營、生活選物、自然材質、溫柔留白
```

## 2. 目標色標系統

### 2.1 核心色票

| 色彩角色 | 色彩名稱 | 色碼 | 主要用途 |
|---|---|---:|---|
| Page Background | 奶油白 | `#FAF9EE` | 頁面主要背景 |
| Surface | 暖瓷白 | `#FFFDF7` | 卡片、表單、彈窗 |
| Surface Soft | 淺奶茶 | `#EFE7DC` | 次要內容區塊 |

| Primary Hover | 森林灰綠 | `#5F705B` | Hover、Active |
| Primary Soft | 淺嫩綠 | `#D9E6D1` | 選中狀態、提示背景 |
| Secondary | 柔霧綠 | `#B6C6AF` | 次要按鈕、圖示背景 |
| Linen | 亞麻米色 | `#DCCFC0` | 邊框、分隔線 |
| Oat | 燕麥色 | `#E8DED1` | 卡片底色、柔和區塊 |
| Text Primary | 深灰綠 | `#3E463D` | 標題、主要文字 |
| Text Secondary | 深奶茶棕 | `#5B5148` | 內文、次標題 |

| Error | 柔霧磚紅 | `#A65F58` | 錯誤、刪除 |
| Error Soft | 淡粉陶土 | `#F4E7E4` | 錯誤提示背景 |

### 2.2 色彩階層

#### 奶油背景階層

```text
#FAF9EE  頁面主背景
#FFFDF7  卡片與表單
#EFE7DC  區塊背景
#E8DED1  裝飾或分類卡片
```

這四個色彩負責建立溫暖感，不應大面積使用純白 `#FFFFFF`。

#### 鼠尾草綠階層

```text
#D9E6D1  淺色背景
#B6C6AF  次要元件
#9FAA96  品牌標準色
#7F927A  中階互動色
#687A64  主要按鈕
#5F705B  Hover / Active
```

不要讓 `#9FAA96` 直接承載白色小文字。它較適合標籤、背景、圖示與裝飾；主要按鈕應使用較深的 `#687A64`。

## 3. 色彩使用比例

建議整體頁面遵循：

```text
60% 奶油白與暖白
25% 奶茶、亞麻、燕麥色
10% 鼠尾草綠
 5% 深色文字與狀態色
```

這樣能避免畫面變成「滿版綠色露營網站」。

### 主要分配

| 區域 | 建議色彩 |
|---|---|
| Body 背景 | `#FAF9EE` |
| Header | `#FFFDF7` 或透明暖白 |
| Hero 文字區 | `#FAF9EE`、`#EFE7DC` |
| 商品卡片 | `#FFFDF7` |
| 分類卡片 | `#EFE7DC`、`#D9E6D1`、`#E8DED1` |
| 主要按鈕 | `#687A64` |
| 次要按鈕 | 暖白背景＋`#9FAA96` 邊框 |
| Footer | `#687A64` |
| Footer 文字 | `#FAF9EE` |
| 邊框 | `#DCCFC0` |
| 主要文字 | `#3E463D` |

## 4. 元件設計定調

### 4.1 按鈕

#### Primary Button

```css
background: #687A64;
color: #FFFDF7;
border: 1px solid #687A64;
```

Hover：

```css
background: #5F705B;
border-color: #5F705B;
```

#### Secondary Button

```css
background: #FFFDF7;
color: #687A64;
border: 1px solid #9FAA96;
```

Hover：

```css
background: #D9E6D1;
```

#### 按鈕外觀

```text
高度：44px–48px
圓角：999px 或 14px
字重：500–600
陰影：原則上不使用，必要時僅使用淡陰影
```

避免：

- 亮橘色 CTA
- 高飽和黃色
- 純黑按鈕
- 強烈漸層
- 厚重投影

### 4.2 卡片

```css
background: #FFFDF7;
border: 1px solid #DCCFC0;
border-radius: 18px;
box-shadow: 0 4px 18px rgba(62, 70, 61, 0.08);
```

Hover：

```css
box-shadow: 0 8px 26px rgba(62, 70, 61, 0.12);
transform: translateY(-3px);
```

卡片不應全部純白。分類卡片和內容卡片可以交替使用：

```text
#FFFDF7
#EFE7DC
#D9E6D1
#E8DED1
```

### 4.3 標籤

#### 新品標籤

```css
background: #D9E6D1;
color: #3E463D;
```

#### 人氣推薦

```css
background: #E8DED1;
color: #5B5148;
```

#### 本週選物

```css
background: #B6C6AF;
color: #3E463D;
```

不要使用紅色或黃色促銷標籤。

### 4.4 表單

```css
background: #FFFDF7;
border: 1px solid #DCCFC0;
color: #3E463D;
```

Focus：

```css
border-color: #9FAA96;
box-shadow: 0 0 0 3px rgba(159, 170, 150, 0.18);
```

Placeholder：

```css
color: #857B72;
```

表單輸入框圓角建議：

```text
12px–14px
```

## 5. 版面定調

### Header



```css
background: rgba(255, 253, 247, 0.94);
border-bottom: 1px solid #E8DED1;
```

導覽文字：

```text
預設：#5B5148
Hover：#687A64
Active：#687A64
```

目前頁面可用細底線或淡綠膠囊標示，不使用大型深色區塊。

### Hero

Hero 應以照片為主，不應使用深綠漸層當作主視覺。

照片色調：

- 奶油白帳篷
- 柔和晨光
- 低對比森林
- 原木與亞麻
- 鼠尾草綠露營用品
- 避免過暗與高對比

遮罩建議：

```css
background: linear-gradient(
  90deg,
  rgba(250, 249, 238, 0.82) 0%,
  rgba(250, 249, 238, 0.58) 38%,
  rgba(250, 249, 238, 0.08) 72%
);
```

文字使用：

```text
主標題：#3E463D
副標題：#5B5148
```

### Footer



```css
background: #687A64;
color: #FAF9EE;
```

次要文字：

```css
color: rgba(250, 249, 238, 0.72);
```

連結 Hover：

```css
color: #FFFDF7;
```

不要使用冷青色或科技藍支付圖示。

- 商品資訊

- 按鈕

## 7. 圓角、陰影與間距

### 圓角

```text
小型元件：10px
輸入框：12px
按鈕：14px 或膠囊形
商品卡片：18px
大型區塊：24px
```

### 陰影

```css
--shadow-soft:
  0 4px 18px rgba(62, 70, 61, 0.08);

--shadow-hover:
  0 8px 26px rgba(62, 70, 61, 0.12);
```

避免：

```text
黑色高透明陰影
多層立體陰影
強烈漂浮感
```

### 間距

建議採用 8px 基準：

```text
8px
16px
24px
32px
48px
64px
96px
```

主要區塊上下距離建議為 `72px–96px`，保持呼吸感。

## 8. 圖像風格

圖片應符合：

```text

低飽和
奶油霧感
自然陰影
原木
亞麻
陶瓷
手沖咖啡
書籍
乾燥花
森林與草地
```

避免：


商品圖片可保留自然環境，不需要每張都使用純白背景。

## 10. 最終設計原則

整個網站應遵循以下一句話：

> 以奶油白建立空間，以亞麻米色增加溫度，以鼠尾草綠建立品牌識別，以深灰綠維持文字可讀性。

視覺上應避免讓任何一個顏色過度突出。使用者第一眼應感受到的是「溫柔、自然、安靜」，第二眼才注意到商品與功能。
