# Yuruicamp｜品牌設計定調 v2.1｜CTA 層級修正版

> 以原有「療癒系文青奶油風 × 日系露營選物 × 北歐自然生活感」為基礎，補上 Yuruicamp 更需要的「鬆弛、親切、新手友善、不必太用力」品牌個性，並建立清楚的 CTA 轉換層級：**綠色負責品牌與信任，陶土橘負責行動與轉換。**

---

## 1. 品牌核心定位

### 品牌一句話

> **不用準備得很完美，也可以舒服地去露營。**

Yuruicamp 不是強調征服自然、裝備競賽或極限冒險的戶外品牌，而是一個讓人用自己的步調接近自然的露營生活品牌。

品牌提供的不只是露營用品，而是：

- 更容易開始的露營方式
- 更舒服的戶外生活選擇
- 不製造裝備焦慮的購物體驗
- 有人陪著挑選、理解與使用裝備的安心感
- 讓第一次露營也能輕鬆完成的小小成就感

### 品牌定位

```text
Soft Outdoor Lifestyle
輕鬆自在的戶外生活品牌
```

整體風格定義：

> **療癒系文青奶油風 × 日系露營選物 × 北歐自然生活感 × 輕鬆新手友善**

核心不是「戶外冒險」，而是：

> **把舒服的日常帶到戶外。**

### 品牌標語

```text
Camp at Your Own Pace.
按照自己的步調去露營。
```

中文品牌主張：

```text
露營，不需要太用力。
```

可交替使用：

```text
把舒服帶到戶外。
慢慢準備，也能好好出發。
第一次露營，也可以很輕鬆。
不需要懂很多，也能開始露營。
週末，慢慢來。
```

## 2. 品牌希望帶給人的感受

Yuruicamp 的視覺與體驗應依照以下優先順序建立：

1. 放鬆
2. 溫暖
3. 搖曳
4. 自然
5. 文青
6. 安心
7. 有一點可愛與幽默

使用者第一眼應感受到：

> 「這裡看起來很舒服，我不需要很專業也可以開始。」

第二眼才注意商品與功能。

### 應出現的感受

```text
慢生活、森林晨光、奶油帳篷、原木、亞麻、手沖咖啡、
週末露營、生活選物、自然材質、柔和留白、新手友善、
不用太認真、小小冒險、有人陪著準備
```

### 不應出現的感受

```text
征服自然、極限冒險、軍事戰術、裝備炫耀、高壓促銷、
黑橘硬派、冷冽科技、昂貴距離感、複雜難懂、只有老手才適合
```

## 3. 品牌人格與語氣

可以把 Yuruicamp 想像成一個人：

> 喜歡咖啡、簡單料理與週末郊外。懂得挑選好用的東西，但不賣弄專業；不追求把行程排滿，也不要求每次露營都完美。朋友第一次露營時，會幫忙準備一份剛剛好的清單，提醒對方帶外套，然後說：「沒關係，慢慢來就好。」

### 品牌人格比例

```text
35% 搖曳鬆弛
25% 親切療癒
20% 日系文青感
10% 專業可靠
10% 可愛幽默
```

品牌不宜變得過度精緻、疏離或像精品選物店。

### 品牌說話方式



避免：

- 「露營必備，不買會後悔」
- 「專業玩家唯一選擇」
- 「頂級機能，征服所有地形」
- 「最後倒數，錯過不再」
- 過度使用驚嘆號、倒數計時與紅色促銷

### 文案範例

```text
第一次露營，不知道從哪裡開始？
我們先幫你準備一套簡單的。


不一定要走很遠，舒服就是目的地。

一張椅子、一盞燈、一杯熱飲，
就是一個剛剛好的晚上。

裝備不用一次買齊，
先帶上真正需要的就好。
```

## 4. 目標使用者

主要受眾：

- 第一次或剛開始接觸露營的人
- 喜歡日系、北歐與自然生活風格的消費者
- 希望用品好看、容易使用、容易收納的人
- 週末輕露營、車露、野餐與家庭露營族群
- 情侶、朋友、小家庭與單人露營者
- 重視生活感而非極限機能的使用者

常見擔心：

```text
不知道要買什麼
怕裝備太多
怕搭設太麻煩
怕第一次準備不完整
怕商品看不懂
怕買錯尺寸或規格
怕價格太高
怕露營很累
```

網站應回答：

> 「不用一次懂全部，我們會一步一步幫你選。」

## 5. 視覺風格定位

### 主風格

```text
Japanese Soft Outdoor Minimalism
日系柔和戶外極簡
```

搭配：

```text
Creamy Editorial
奶油系生活編輯感
```

整體應像：

> 森林旁的一間小型露營生活選物店，有晨光、咖啡香、亞麻布與一點不刻意的手繪痕跡。

### 視覺比例

```text
55% 自然戶外
25% 日系生活選物
10% 北歐簡約
10% 輕鬆手繪與品牌個性
```

### Yuruicamp 專屬的「鬆弛感」

在原本精緻、安靜的視覺基礎上，加入：

- 輕微不對稱的插畫構圖
- 帶手繪感的帳篷、杯子、雲、營火與樹木線條
- 小幅度歪斜或錯位的裝飾
- 像手寫備註的短句
- 柔和而非完美幾何的圖形
- 「慢慢來」「今天也不用趕」等小文案
- 有一點慵懶表情、但不幼稚的角色或圖示

> 鬆弛感不是凌亂，而是在清楚秩序中保留一點不刻意。

## 6. 目標色標系統

### 色彩核心概念

> **奶油白建立空間，亞麻米色增加溫度，鼠尾草綠建立品牌辨識，深灰綠維持可讀性，營火陶土橘負責重要行動。**



```text
奶油鼠尾草綠色：品牌識別、信任、瀏覽與一般互動
陶土橘：加入購物車、預訂、結帳、送出等轉換行動
奶油與亞麻色：承載內容與建立空間
```

陶土橘不是另一個大面積主題色，而是全站固定、少量且高辨識的轉換色。

| Surface Soft | 淺奶茶 | `#EFE7DC` | 次要內容區塊 |
| Oat | 燕麥色 | `#E8DED1` | 分類卡片、裝飾區塊 |

| Primary | 鼠尾草綠 | `#9FAA96` | 品牌識別、標籤、裝飾 |
| Primary Medium | 中階鼠尾草綠 | `#7F927A` | 瀏覽型按鈕、一般互動 |
| Primary Strong | 深鼠尾草綠 | `#687A64` | 品牌重點、次要實心按鈕、Footer |

### 色彩使用比例

```text
60% 奶油白與暖白
23% 奶茶、亞麻、燕麥色
10% 鼠尾草綠
 4% 深灰綠文字與功能色
 3% 營火陶土橘 CTA 與狀態色
```

陶土橘雖只占約 3%，但因色相與鼠尾草綠有明顯差異，已足以形成視覺焦點。CTA 的效果來自「少量、固定、只出現在重要行動」，而不是大面積使用。

### 陶土色角色分工

陶土色分為兩種用途，不能混用：

#### 轉換 CTA 色

```text
#A85D3A  營火陶土橘
#8F4E31  CTA Hover
#7E432B  CTA Active
#F1E2D8  CTA Soft
```

用於：

- 開始挑選裝備
- 加入購物車
- 立即預訂
- 確認預訂
- 前往結帳
- 完成付款
- 送出租借申請
- 訂閱電子報

#### 插畫與人情溫度色

```text
#A86F5B  柔和陶土
```

只用於：

- 營火、杯子等手繪插畫
- 小型品牌裝飾
- 重要但非危險的提示圖示
- 不超過整頁約 1%–2% 的細節

禁止：

- 陶土橘大面積鋪底
- Header 或 Footer 全面改成橘色
- 所有按鈕都使用陶土橘
- 標籤、圖示與裝飾大量搶用 CTA 色
- 使用高飽和亮橘、螢光橘或偏黃色的橘色

## 7. UI 元件設計定調

### 7.1 第一層：Conversion CTA

只用於使用者即將採取的重要轉換行動：

- 開始挑選裝備
- 加入購物車
- 立即預訂
- 確認預訂
- 前往結帳
- 完成付款
- 送出申請
- 訂閱電子報

```css
.btn-cta {
  background: #A85D3A;
  color: #FFFDF7;
  border: 1px solid #A85D3A;
}
```

Hover：

```css
.btn-cta:hover {
  background: #8F4E31;
  border-color: #8F4E31;
  transform: translateY(-1px);
}
```

Active：

```css
.btn-cta:active {
  background: #7E432B;
  border-color: #7E432B;
  transform: translateY(0);
}
```

Focus：

```css
.btn-cta:focus-visible {
  outline: none;
  box-shadow: 0 0 0 3px rgba(168, 93, 58, 0.22);
}
```

### 7.2 第二層：品牌與瀏覽型按鈕

用於不會立即產生交易的操作：

- 查看商品
- 查看推薦組合
- 閱讀新手指南
- 探索露營日誌
- 修改內容
- 查看詳細資訊

實心版本：

```css
.btn-brand {
  background: #687A64;
  color: #FFFDF7;
  border: 1px solid #687A64;
}
```

Hover：

```css
.btn-brand:hover {
  background: #5F705B;
  border-color: #5F705B;
}
```

### 7.3 第三層：Secondary Button

適合「先看看」「返回」「修改」「閱讀更多」：

```css
.btn-secondary {
  background: #FFFDF7;
  color: #687A64;
  border: 1px solid #9FAA96;
}
```

Hover：

```css
.btn-secondary:hover {
  background: #D9E6D1;
}
```

### 7.4 第四層：Tertiary Action

用於取消、稍後決定、查看更多與文字型操作：

```css
.btn-tertiary {
  background: transparent;
  color: #5B5148;
  border: none;
  text-decoration: underline;
  text-underline-offset: 4px;
}
```

### 7.5 按鈕文字與尺寸

```text
高度：44px–48px
圓角：14px 或膠囊形
字重：500–600
主要 CTA 可比次要按鈕略寬或略高
同一區塊最多只能有一個陶土橘主要 CTA
```

推薦文字：

```text
第一次露營從這裡開始
開始挑選裝備
加入購物車
立即預訂
確認預訂
前往結帳
看看推薦組合
閱讀新手指南
```

避免：

```text
立即搶購
錯過不再
終極方案
最後倒數
```

### 7.6 商品卡 CTA 規則

首頁為了維持安靜感，商品卡平常可使用綠色外框；Hover 或使用者聚焦時，再轉為陶土橘：

```css
.product-card .add-cart {
  background: #FFFDF7;
  color: #687A64;
  border: 1px solid #9FAA96;
}

.product-card:hover .add-cart,
.product-card .add-cart:focus-visible {
  background: #A85D3A;
  color: #FFFDF7;
  border-color: #A85D3A;
}
```

正式商品列表、商品詳情、購物車與結帳頁，可直接使用陶土橘「加入購物車／預訂／結帳」按鈕。

### 7.7 CTA 不只靠顏色

- 一個區塊最多只有一個主要 CTA
- 主要按鈕周圍應有足夠留白
- 主要按鈕尺寸略高於次要按鈕
- 文案需明確說明點擊後會發生什麼
- 不讓標籤、插畫與一般圖示搶用 CTA 色
- Hover、Focus、Active 狀態必須清楚
- 陶土橘只出現在真正重要的下一步

### 卡片

```css
background: #FFFDF7;
border: 1px solid rgba(220, 207, 192, 0.72);
border-radius: 18px;
box-shadow: 0 3px 14px rgba(62, 70, 61, 0.045);
```

Hover：

```css
box-shadow: 0 8px 24px rgba(62, 70, 61, 0.10);
transform: translateY(-2px);
```

### 標籤

```text
新手友善：#D9E6D1 / #3E463D
輕鬆好收：#E8DED1 / #5B5148
本週選物：#B6C6AF / #3E463D
一個人也適合：#EFE7DC / #5B5148
```

避免紅色、亮黃色與大量「熱賣」「搶購」角標。

### 表單

```css
background: #FFFDF7;
border: 1px solid #DCCFC0;
color: #3E463D;
border-radius: 12px;
```

Focus：

```css
border-color: #9FAA96;
box-shadow: 0 0 0 3px rgba(159, 170, 150, 0.18);
```

錯誤訊息應使用溫和語氣：

```text
這裡好像少填了一點資料。
再確認一次日期就可以囉。
```

## 8. 版面設計定調

### Header

使用暖白或半透明暖白，不使用深色 Header。

```css
background: rgba(255, 253, 247, 0.94);
border-bottom: 1px solid #E8DED1;
```

導覽文字：

```text
預設：#5B5148
Hover：#687A64
Active：#687A64
```

### Hero

Hero 以生活場景照片為主，不使用大面積深綠色主視覺。

照片內容：

- 奶油白帳篷
- 柔和晨光
- 低對比森林
- 原木與亞麻
- 咖啡、書、毯子、燈具
- 人物正在準備、聊天、閱讀或休息
- 不需要過度完美的擺拍

遮罩：

```css
background: linear-gradient(
  90deg,
  rgba(250, 249, 238, 0.84) 0%,
  rgba(250, 249, 238, 0.60) 38%,
  rgba(250, 249, 238, 0.10) 72%
);
```

Hero 文案：

```text
把日子，慢慢搬進森林。

不需要準備得很完美，
也可以舒服地去露營。
```

CTA：

```text
第一次露營從這裡開始  → 陶土橘實心 CTA
看看適合我的裝備      → 暖白底＋鼠尾草綠外框
```

Hero 中只能有一個陶土橘主按鈕，其餘操作維持綠色外框或文字形式。

### Footer

Footer 是唯一可以大面積使用深鼠尾草綠的區域。

```css
background: #687A64;
color: #FAF9EE;
```

可加入手繪帳篷、雲、冒著蒸氣的杯子，以及：

```text
今天也不用趕。
```

## 9. 新手友善體驗

首頁至少保留以下三種入口：

```text
第一次露營從這裡開始
三步找到適合你的裝備
一人、兩人、家庭露營組合
租借還是購買？
不知道怎麼選？我們幫你配
週末輕露營清單
```

商品卡片優先顯示：

- 適合人數
- 搭設難度
- 收納大小
- 新手友善程度
- 適合情境
- 是否可租借
- 建議搭配商品

範例：

```text
適合 1–2 人
約 10 分鐘完成搭設
新手友善
轎車也放得下
```

推薦組合命名：

```text
第一次兩人露營
一個人的安靜週末
小家庭輕鬆出發
咖啡與營火下午
不用搭太久的露營組
```

避免：

```text
A方案
高階方案
尊榮套組
旗艦級組合
```

導購語氣：

```text
不確定怎麼選？
先告訴我們人數和露營方式，
剩下的慢慢來就好。
```

## 10. 品牌圖形與插畫

### Logo 方向

建議：

```text
yuruicamp
```

或：

```text
YURUI CAMP
SOFT OUTDOOR GOODS
```

建議特徵：

- 圓潤字形
- 字距略寬
- 小寫形式較親切
- 可加入輕微手寫或不完全對稱的細節
- 圖形需能使用在吊牌、杯子與社群頭像

### 專屬品牌符號

推薦：

- 慵懶的小帳篷
- 帳篷上方的咖啡蒸氣
- 慢慢飄過的雲
- 微微發光的營燈
- 坐在帳篷前休息的小角色
- 歪一點但舒服的手繪營火
- 緩慢彎曲的小路

最推薦：

> **帳篷＋蒸氣／雲朵**

### 插畫風格

```text
單色或雙色線稿
細線條
略帶手繪感
不完全對稱
留有空白
低對比
不追求精準透視
```

避免動漫式大型角色、複雜山脈徽章、斧頭刀具與軍事符號。

## 11. 字體、圓角、陰影與圖片

### 字體

```css
--font-heading: "Noto Serif TC", serif;
--font-body: "Noto Sans TC", sans-serif;
```

字重：

```text
主標題：600
區塊標題：600
內文：400
按鈕：500–600
```

手寫感字體只可少量用於備註、插畫旁短句或社群素材。

### 圓角

```text
小型元件：10px
輸入框：12px
按鈕：14px 或膠囊形
商品卡片：18px
大型區塊：24px
```

### 陰影

```css
--shadow-soft: 0 3px 14px rgba(62, 70, 61, 0.045);
--shadow-hover: 0 8px 24px rgba(62, 70, 61, 0.10);
```

### 圖片風格

符合：

```text
柔和晨光、低飽和、奶油霧感、自然陰影、原木、亞麻、
陶瓷、手沖咖啡、書籍、毯子、森林與草地、真實使用狀態
```

人物狀態：

- 坐著休息
- 慢慢準備餐點
- 閱讀
- 喝咖啡
- 和朋友聊天
- 不需要看鏡頭

避免：

```text
高反差 HDR、鮮豔橘色帳篷、軍綠迷彩、黑色戰術裝備、
極限登山、冷色棚拍、完美到不真實的擺拍
```

## 12. 電商操作原則

Yuruicamp 可以柔和，但不能模糊。

- 主要轉換 CTA 使用 `#A85D3A`
- CTA Hover 使用 `#8F4E31`
- CTA Active 使用 `#7E432B`
- 品牌與瀏覽型按鈕使用 `#687A64`
- 綠色 Hover 使用 `#5F705B`
- 選中狀態使用 `#D9E6D1`
- Focus 狀態必須清楚
- 價格、庫存、日期與規格不可因風格而降低可讀性
- 加入購物車、預訂與結帳不可和一般「查看詳情」使用相同層級

商品卡層級：

```text
情境標籤
商品圖片
商品名稱
適合對象／使用難度
價格
主要行動
收藏
```

促銷文案推薦：

```text
本週慢慢選
週末剛好適合
新手組合少一點準備
一起帶走比較省事
```

避免：

```text
最後一小時
瘋狂下殺
買到賺到
倒數搶購
紅色滿版折扣
```

## 15. 禁止事項

### 色彩

禁止：

- 高飽和黃色
- 高飽和亮橘、螢光橘或偏黃橘色 CTA
- 大面積深咖啡色
- 黑橘戶外配色
- 螢光綠
- 冷科技藍
- 三種主題色平均分配
- 每個圖示使用不同顏色

### 版面

禁止：

- 深綠色滿版 Hero
- 過多促銷模組
- 元件擁擠
- 大量粗框線
- 高對比陰影
- 所有區塊都使用卡片包住
- 只有漂亮照片，卻沒有新手引導

### 文案

禁止：

- 製造焦慮
- 賣弄術語
- 對新手說教
- 過度高級、疏離
- 太多英文戶外術語
- 把露營說成需要完整裝備才能開始

## 16. 最終檢查清單

### 品牌感受

- [ ] 第一眼是否溫柔、自然、安靜？
- [ ] 是否能感受到「慢慢來」？
- [ ] 是否有親切感，而不只有高級感？
- [ ] 新手是否會覺得自己也適合？
- [ ] 是否避免裝備焦慮與專業距離感？

### 色彩

- [ ] 奶油中性色是否占主要比例？
- [ ] 鼠尾草綠是否建立清楚品牌識別？
- [ ] 是否沒有大面積黃色與深咖啡色？
- [ ] 陶土橘是否只用於重要轉換行動，而沒有被裝飾與標籤濫用？
- [ ] 綠色按鈕與陶土橘 CTA 是否有清楚的功能分工？
- [ ] 照片是否與 UI 色彩一致？

### UI

- [ ] 主要 CTA 是否清楚？
- [ ] Focus、Hover、選中狀態是否可辨識？
- [ ] 商品資訊是否比裝飾更重要？
- [ ] 是否提供新手入口與簡單導購？
- [ ] 柔和風格是否沒有犧牲操作性？

### Yuruicamp 專屬個性

- [ ] 是否有至少一個專屬品牌圖形？
- [ ] 是否加入一點手繪與鬆弛感？
- [ ] 文案是否像朋友，而不是銷售員？
- [ ] 是否有「不用完美也可以開始」的訊息？
- [ ] 整體是否不像通用日系模板？

## 17. 最終設計原則

> **以奶油白建立空間，以亞麻米色增加溫度，以鼠尾草綠建立品牌識別，以深灰綠維持文字可讀性，以營火陶土橘指出重要的下一步。**

> **在安靜精緻的視覺中，加入一點鬆、一點親切，以及第一次露營也能安心開始的陪伴感。**

> **綠色讓使用者認得品牌，陶土橘讓使用者知道下一步。**

最終想呈現的不是：

> 在森林裡，完美地過一段精緻生活。

而是：

> **不需要準備得很完美，也可以用自己的步調，舒服地去露營。**
