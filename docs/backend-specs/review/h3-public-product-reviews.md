# H-3 商品公開評論

- 正式 PostgreSQL 評論與照片讀取
- `page`、`size` 分頁
- 最新、最高分、最低分排序
- 精確星等與是否包含照片篩選
- 評論總數、平均評分與各星等數量
- 商品不存在時回傳 `404 NOT_FOUND`
- 公開 DTO 排除會員與訂單識別資訊

資料關聯以 `reviews.order_item_id`、`order_items.product_id` 為準；商品存在性沿用公開商品目錄條件。

處理流程：

1. 驗證商品可公開販售及排序參數。
2. 查詢不受篩選影響的商品整體評分統計。
3. 使用 `rating`、`hasPhotos` 計算篩選後筆數並讀取指定頁面。
4. 批次讀取該頁評論照片，組合不含會員與訂單識別資訊的公開 DTO。
5. `data.summary` 回傳整體統計，`meta` 回傳篩選後分頁資訊。
