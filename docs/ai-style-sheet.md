# Yuruicamp AI 樣式規範

**狀態：** 草稿
**最後更新：** 2026-06-30
**來源：** 現有的 Yuruicamp HTML／SCSS／訂房 CSS，以及 `ui-ux-pro-max` 的無障礙與響應式規則。

---

## 目的

這份樣式規範保留為主站歷史參考文件；新的設計 token source of truth 為 `booking/booking-style-tokens.md` 的 `--yc-*` 規格。
主站與 booking 目前統一使用 `--yc-*`，不再輸出 `--yui-*` 或 `--bk-*` 相容 alias。

## 設計方向

* 產品類型：戶外電商、營地訂房、租借結帳、賣家後台管理。
* 視覺氛圍：平靜、自然、實用、可靠、容易瀏覽。
* 建議風格：節制的工具型 UI，搭配戶外編輯風格的溫度。
* 避免使用：強烈漸層、裝飾性 blob 圖形、以 emoji 作為 UI 圖示、一次性的顏色、流程頁面中過大的行銷區塊。
* 新增 UI 時，請使用同一套 SVG 圖示庫的圖示。既有的 emoji 類內容若屬於文章或行銷內容可以保留，但新的互動控制項必須使用圖示。

## 核心 Token

核心 token 已移至 `booking/booking-style-tokens.md`，本文件只保留設計語氣與元件規則。實作時使用 `--yc-*`，不要新增 `--yui-*` 或 `--bk-*`。

## 字體排版

* 基礎字體：使用與現有 SCSS 一致的系統 UI 字體堆疊。
* 內文字體大小：最小為 `16px`。
* 內文行高：`1.5` 到 `1.8`。
* 標題：使用 `700` 字重，顏色使用深綠或接近黑色。
* 資料密集的後台表格：可使用 `14px` 到 `15px`，但可點擊元素仍應保持至少 `44px` 高的觸控目標。

## 版面規則

* 前台網站主容器：`min(1120px, calc(100vw - 32px))`。
* 商品與文章網格：`repeat(auto-fit, minmax(240px, 1fr))`。
* 結帳與詳細頁面：桌面版使用雙欄版面；低於 `768px` 時折疊為單欄。
* 訂房頁面：摘要面板僅在桌面版使用 sticky；手機版摘要必須回到一般文流中。
* 後台頁面：優先考慮密集掃讀、表格、篩選器與 Modal；避免使用行銷風格卡片。

## 元件規則

### 按鈕

* 主要操作：深綠背景、白色文字、`8px` 圓角、最小高度 `44px`。
* 次要操作：白色或淡綠色背景、主要色邊框、主要色文字。
* 破壞性操作：僅在操作不可逆時使用危險色文字或危險色背景。
* 載入中的操作：停用點擊功能、維持按鈕寬度穩定、顯示文字或 loading spinner。

### 卡片

* 圓角：商品、文章、後台面板使用 `8px`；Hero 區或大型訂房卡片才可使用 `12px`。
* 陰影：保持細微，不使用強烈發光效果。
* Hover：改變邊框或陰影；不要用縮放卡片的方式造成版面跳動。

### 表單

* 每個輸入欄位都必須有可見標籤；若標籤文字視覺上隱藏，則必須提供關聯的 `aria-label`。
* 錯誤訊息必須放在欄位附近。
* 付款與結帳區塊應持續顯示使用者目前選擇的值。

### 導覽

* 頁首控制項必須依照視覺順序維持鍵盤可達性。
* 僅有圖示的按鈕必須提供 `aria-label`。
* Offcanvas 與 Modal 開啟時必須鎖定焦點；關閉後必須將焦點還原至原本觸發的元素。

### 狀態與標籤

* 狀態顏色必須搭配文字標籤。
* 訂單、訂房、租借或評價狀態不可只依賴顏色來區分。

## AI 實作 Prompt

生成此專案 UI 時，請使用以下指令：

`先閱讀 docs/ai-style-sheet.md 與 docs/ai-style-tokens.css。保留現有 Yuruicamp 的綠色戶外品牌識別。使用 docs/frontend-specs/pages/[page].md 中現有的頁面規格。除非規格明確要求，否則不要引入新的視覺系統。`

## 驗收檢查清單

* [ ] 使用 Yuruicamp token 名稱，而不是硬編碼新的顏色。
* [ ] 手機版內文字體至少維持 `16px`。
* [ ] 所有互動控制項都有可見的 hover 與 focus 狀態。
* [ ] 最小觸控目標為 `44px`。
* [ ] 版面可在 `375px`、`768px`、`1024px` 與 `1440px` 正常運作。
* [ ] 動畫時間使用 `150ms` 至 `300ms`，並遵守 `prefers-reduced-motion`。
* [ ] 新增的互動控制項不可使用 emoji 圖示。
