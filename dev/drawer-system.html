Yuruicamp Drawer System Sandbox

統一滑出式面板設計系統驗證頁

以裝備商城選單為基準，統整導覽、登入與購物車三種 Drawer 的外殼、節奏與視覺語言。

Desktop 1440 × 900 Tablet 1024 × 768 Portrait 768 × 1024 Mobile 390 × 844

設計原則

共用外殼、分離內容、維持低飽和與柔和層級。

外殼一致

Header、Body、Footer、Overlay 與 Close button 只保留一套。

內容差異化

導覽、登入、購物車僅透過內容與少量 modifier 區隔。

品牌低彩度

以鼠尾草綠、奶油白、柔和邊框與輕量陰影作為主要語言。

可及性

支援鍵盤、Escape、Focus return 與清楚的 aria 標記。

Drawer Width & RWD

桌機維持邊距與完整圓角，手機縮為近全寬但不貼邊。

Desktop

  • 最大寬度約 480px
  • 視窗右側與上下保留 16px
  • 圓角完整可見

Mobile

  • 約 `calc(100vw - 1rem)`
  • 保留 8–12px 外邊距
  • Body 可獨立捲動
