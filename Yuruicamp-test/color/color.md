# Yuruicamp 露營選物品牌網站 - 配色方案

## 核心主色

| 色系 | 顏色代碼 | 應用場景 |
|------|---------|---------|
| **深青綠色** | `#244d4d` | Header、Logo、按鈕、邊框、主文字 |
| **純白色** | `#fff` | 背景、按鈕文字、卡片背景 |
| **淺青灰綠** | `#779` | 連結、副標題、Hover 效果 |
| **淺綠背景** | `#f6fbf6` | Header 背景 |

## 強調色

| 色系 | 顏色代碼 | 用途 |
|------|---------|------|
| **黃色** | `yellow` | 文字強調、連結 Hover |
| **金色** | `gold` | 星級評分圖示 |
| **淺灰色** | `#f2f2f2` | 商品卡片背景 |
| **深青綠懸停色** | `#316868` | 按鈕 Hover 效果 |

## SCSS 變數定義

```scss
$green: #244d4d;           // 主色
$white: #fff;              // 輔助色
$light-color: #779;        // 次要色
$light-bg: #f6fbf6;        // 背景色
$border: 0.1rem solid $green;
$box-shadow: 0.5rem 1rem rgba(0, 0, 0, 0.1);
$dark-shadow: 0 0 0 100vw rgba(0, 0, 0, 0.5);
```

## 品牌色彩理念

- **主題色 `#244d4d`**：深青綠色傳達自然、沉穩、戶外的品牌形象
- **文字層次**：主文字用深青綠，次要文字用淺青灰綠
- **互動反饋**：Hover 時深化或轉換色彩
- **卡片設計**：白底 + 淺灰背景突顯商品

適合露營戶外電商的專業、可信賴視覺風格。
